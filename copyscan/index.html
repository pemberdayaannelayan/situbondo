Pemindai Excel Cerdas

Deteksi dan gabungkan data duplikat berdasarkan NIK dengan alamat terlengkap

Fitur Aplikasi: Unggah file Excel dengan kolom: Nama, NIK, Alamat, Kecamatan, Desa. Aplikasi akan memindai data duplikat berdasarkan NIK dan mempertahankan data dengan informasi alamat paling lengkap. Tampilan otomatis menyesuaikan dengan ukuran layar perangkat.

Unggah File Excel

Pilih file Excel (.xlsx, .xls) dengan data yang berisi kolom: Nama, NIK, Alamat, Kecamatan, Desa

Belum ada file dipilih

Catatan: Pastikan file Excel memiliki kolom dengan nama tepat: nama, nik, alamat, kecamatan, desa (tidak case-sensitive). Data dengan alamat lebih lengkap akan diprioritaskan saat ada duplikasi NIK.
