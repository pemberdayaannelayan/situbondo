  • Pastikan koneksi internet stabil untuk pengiriman WhatsApp

Backup & Restore Data

Kelola cadangan data dan restore data dari file backup

Backup Data
Buat Backup Data

Simpan salinan data Anda dalam format file JavaScript (reload.js)

Mode Backup: Data akan disimpan berdasarkan mode wilayah saat ini.
Informasi Backup:
  • Data akan disimpan dalam format JavaScript
  • File dapat digunakan untuk restore atau sinkronisasi
  • Nama file disesuaikan dengan mode wilayah
  • Backup data secara berkala untuk menghindari kehilangan data
Restore Data
Restore Data dari File

Upload file backup untuk memulihkan data sebelumnya

Peringatan: Restore akan menggabungkan data lama dengan data baru. Data duplikat akan difilter.
File harus dalam format .js (reload.js) atau .json
Informasi Restore:
  • Hanya file dengan format valid yang dapat di-restore
  • Data duplikat akan difilter otomatis
  • Data lama tetap tersimpan dan digabung dengan data baru
  • Selalu backup data sebelum melakukan restore
