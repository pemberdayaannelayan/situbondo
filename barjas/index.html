    Pratinjau Data Peta
    Panduan Penggunaan
    Langkah-langkah penggunaan data peta:
    1. Masukkan kode keamanan yang telah diberikan
    2. Klik tombol "Generate Data Peta"
    3. Preview data akan muncul, periksa kelengkapan data
    4. Klik "Download Data Peta" untuk mendapatkan file script
    5. File datapeta.js dapat diupload ke repository untuk digunakan di website peta statistik
    Penting: File datapeta.js berisi data statistik per desa yang sudah diringkas.
